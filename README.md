# flask_simple_app
	Description:
		Example of simple app for manage users using flask

	utilities:
		- login validation
		- session validation
		- role validation
		- password (pass + salt) salt and password are sha512

	Install:
		pip install flask
		python main.py # go to http://localhost:5000/index
